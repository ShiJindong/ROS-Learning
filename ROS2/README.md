# ROS2-Learning
---
Used ROS-Version: Foxy

At first:

      cd <name>_ws
      colcon build
      source install/setup.bash
      
Then  
      
      ros2 run "rosnode"

The rosnodes "village_interfaces", "village_wang", "village_zhang" are implemented with C++.

The rosnode "village_li" is implemented with Python.
